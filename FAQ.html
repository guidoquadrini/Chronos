Q: Is ChronosXP really free?

A: Yes.  ChronosXP is free and will remain free. If you like the program we ask that you support the project by making a donation.  Donations can be made at:  http://sourceforge.net/project/project_donations.php?group_id=111901

 

Q: Why should I donate to the project if the program is already free?

A: Donations help to go toward the many expenses in developing and maintaining a project like ChronosXP.  But more importantly, on a spiritual level we know that the more you give the more you get– so donating to the ChronosXP project helps you to get more out of your use of the program!

 

Q: Can I get the source code?

A: Yes, the source code is available at: http://sourceforge.net/projects/chronosxp

 

Q: What are the minimum requirements to run ChronosXP 4.0?

A: Windows XP or later, 2GHz or faster processor, 1GB RAM or more.

 

Q: What languages is ChronosXP available in?

A: ChronosXP is available in English, Dutch, French, Italian, Spanish, Portuguese, Hungarian and Greek.  A partial translation exists for Hebrew.  If you plan to enable Hebrew, make sure you install support for right-to-left languages in Windows.

 

Q: Is ChronosXP available for iPhone/Palm/some other portable device?

A: No.  However it is completely within your power to port ChronosXP to one of these devices, as the source code is freely available.

 

Q: I see two versions of ChronosXP to download, 32-bit and 64-bit.  Which is right for me?

A: If you’ve purchased your computer since the middle of 2008 then you probably want the 64-bit version; computers prior to this date will probably need the 32-bit version.  If you are in doubt as to which version to use, download the 32-bit version – it works on both.

 

Q: When I try to install ChronosXP, I get the error message “This installation package is not supported by this processor type.  Contact your product vendor.”

A: You probably tried to install the 64-bit version of ChronosXP, but you only have 32-bit Windows.  Just because you have a 64-bit processor does not necessarily mean that you have 64-bit Windows.  Try downloading and installing the 32-bit version of the program instead.

Q: I use Hebrew letters instead of the default symbols; how do I enable the alternate Hebrew letters to be displayed?

A: Make sure that you’ve chosen Hebrew as the display type (in the Properties/Display tab), and create an empty file called Alt.txt that is in the same directory as your ChronosXP.exe (usually C:\Program Files\ChronosXP or C:\Program Files (x86)\ChronosXP).  Make sure to re-start the program, and you should see the alternate letters.
